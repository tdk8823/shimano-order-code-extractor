# shimano-order-code-extractor

## setup environment

create venv
```bash
python -m venv venv
```

activate venv
```bash
source venv/bin/activate
```

install packages from requirements.txt
```bash
pip install -r requirements.txt
```

## usage
```bash
python extract_tables.py --input_path path/to/pdf -output_dir path/to/output/dir
```
